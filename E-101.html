CT-101

Vehicle Trackers


CT-101 is a compact vehicle tracker designed for all types of vehicles like personal cars, taxis & commercial transports. It has one digital input which can be used for ignition, door, or alarm button status monitoring & one digital output which can help in achieving vehicle remote immobilization.


Features

Internal Battery Backup

Ignition Sense

Remote Immobilization

Voltage Surge Protection

In-Built Large Memory

Precise GPS Accuracy

Remote Data logging
